# Python Command Line Applications CRUD Menu.

## About 

This project uses command line with Python. It is structured in such a way that 
when we execute the main.py file, we can use a menu CRUD of Product. 

## Setup

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```
